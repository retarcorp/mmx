Конструктор находится по пути /frontend/views/site/constructor.php

Контроллер который отвечает за вывод конструктора нахоидтся по пути /frontend/controllers/SiteController.php
и называется actionConstructor(), там просто указывается какой html нужно подрендерить

Всю информацию по конструтору можно получить в ApiController.php в методе actionItems()

Пример GET запроса на получение всей информации о конструкторах http://mmx.virtual/api/items
В случае если нет ничего, вернется not found

Если же отсутствует продукт с таким артиклем которй есть в контсрутроре в ответе будет только файл конфигурации,
значение артиклся и описание продукта  - null

Если все корректно то вернется массив со всеми конструкторами

Параметр calc это название json файла в котором более детальная информация

Для доступа к этой информации тоже GET запрос - mmx.virtual/api/items?calc=3952sz.json

если отсутствует такой json файл вернется null
